Como Começar
3 passos para minerar
1

Instale o Node.js

Se ainda não tem, baixe em nodejs.org (versão 18 ou superior). É grátis.

node --version # deve aparecer: v18.x.x ou maior
2

Baixe o CEREBRO

Clona o repositório do GitHub e instala as dependências:

git clone https://github.com/ricardocbernardi-ship-it/cerebro-blockchain cd cerebro-blockchain npm install
3

Inicie e comece a minerar

Rode o nó e o minerador automático:

# Inicia a blockchain node no-completo.js # Em outro terminal — minera automaticamente node scripts/auto-miner.js
Dica: Use PM2 para manter rodando 24h mesmo fechando o terminal:
npm install -g pm2 && pm2 start no-completo.js --name cerebro
✓

Veja seu saldo crescer

Abra o Explorer e acompanhe seus CBR chegando em tempo real:

http://localhost:3001/explorer.html
CBR ganhos ficam na sua carteira para você usar, guardar ou vender quando a rede crescer.

Perguntas Frequentes

Preciso de placa de vídeo (GPU)? ▾
Não. CEREBRO usa CPU (processador normal). Qualquer computador com Node.js consegue minerar. Não precisa de hardware especial.
CBR tem valor real agora? ▾
Ainda não tem preço em exchange. Mas o supply é limitado a 21 milhões (igual ao Bitcoin). Quem minera agora está acumulando enquanto ainda é fácil. O valor vem com a adoção da rede.
Onde vou vender meu CBR? ▾
O objetivo é listar o CBR em exchanges. O roadmap inclui listagem em exchanges menores (BitMart, MEXC, XT.com) à medida que a rede cresce. Enquanto isso, CBR pode ser negociado diretamente entre usuários.
Quanto gasto de luz minerando? ▾
Muito pouco. CEREBRO usa CPU normal, não GPU. O consumo é similar a deixar um site aberto no browser. Bem diferente do Bitcoin que usa hardware especializado (ASIC).
Posso rodar em VPS ou servidor na nuvem? ▾
Sim! Você pode rodar em qualquer servidor Linux (DigitalOcean, AWS, Hetzner etc). Um VPS de $5/mês já basta para minerar 24h por dia. Isso maximiza seus ganhos.
O que é staking de CBR? ▾
Staking é "guardar com juros". Você trava seus CBR na blockchain e recebe recompensas extras. É como deixar dinheiro num banco que paga rendimento, mas sem banco — tudo on-chain.
