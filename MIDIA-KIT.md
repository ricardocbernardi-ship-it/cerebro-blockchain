Título: I built a blockchain where smart contracts are written in regular JavaScript

No Solidity. No Rust. Just JS.

function votar(candidato) {
  state.votos[candidato] = (state.votos[candidato] || 0) + 1;
  emit('Voto', { candidato });
  return state.votos;
}

That's a real deployed smart contract on CEREBRO Blockchain.

The contract runs in a Node.js vm.runInNewContext() sandbox — isolated, secure, no file system access.

Full SDK available: const cerebro = require('./cerebro.js')
